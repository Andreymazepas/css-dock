CSS only dock

This is a CSS only dock. It's a quick project to play with CSS's new :has selector and 98.css library.

This selector is not supported by all browsers yet is supported by most browsers now.

The way it works is that it provides an easy way to select a parent element or a previous sibling, in the case of the dock, I'm checking if each element has a hovered sibling.

.dock-program:has(+ .dock-program:hover) {
transform: scale(1.3) translateY(-15%);
}

Later on I'm planning to make this dock usable and configurable to use as a startpage , so stay tuned.
